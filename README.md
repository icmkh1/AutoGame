# AutoGame

一个基于 Python 的键鼠自动化工具，支持多种宏功能类型，可通过 JSON 配置文件进行配置。系统支持热键触发、图像匹配、颜色匹配、文字识别等功能。

## 功能特性

- **多种宏功能类型**：支持连击、固定连击、宏、有序宏、跟随、组合、映射、图像匹配、颜色匹配、文字识别等 10 种功能类型
- **热键触发**：支持自定义触发键，一键启动/停止宏功能
- **图像匹配**：支持在指定区域内查找目标图像
- **颜色匹配**：支持检查指定坐标的颜色是否匹配
- **文字识别**：支持在指定区域内识别文字
- **可视化配置**：提供 Web 界面进行配置文件编辑和管理
- **系统托盘**：最小化到系统托盘，不占用任务栏空间

## 技术栈

- **后端**：Python 3.10+
- **前端**：Vue 3 + TypeScript + Vite
- **键鼠操作**：autoxkit
- **图像匹配**：autoxkit
- **文字识别**：ONNX PaddleOCR
- **GUI框架**：PyWebView

## 安装与运行

### 环境要求

- Python >= 3.10
- Node.js >= 18.0.0

### 安装步骤

1. **克隆项目**

```bash
git clone https://github.com/YorickFin/AutoGame.get
```

1. **安装后端依赖**

```bash
uv sync
```

1. **安装前端依赖**

```bash
cd frontend
pnpm install
```

### 开发模式运行

1. **启动前端开发服务器**

```bash
cd frontend
pnpm run dev
```

1. **启动后端程序**

```bash
uv run main.py
```

### 打包部署

1. **构建前端**

```bash
cd frontend
pnpm run build
```

1. **打包为可执行文件**

```bash
pyinstaller main.spec -y
```

## 使用说明

### 配置文件格式

配置文件采用 JSON 数组格式，包含一个基本信息配置项和多个宏功能配置项。

**基本信息配置：**

```json
{
    "备注": "基本信息",
    "按键更改": "",
    "坐标更改": "(2560,1440)",
    "鼠标图标更改": "是"
}
```

**宏功能配置示例：**

```json
{
    "功能类型": "连击",
    "触发键": "P",
    "每秒次数": "30",
    "宏指令": "D"
}
```

### 支持的功能类型

| 功能类型 | 说明                  |
| ---- | ------------------- |
| 连击   | 按住触发键时持续重复执行宏指令     |
| 固定连击 | 执行指定次数的连击，可配置后置指令   |
| 宏    | 单次执行宏指令             |
| 有序宏  | 按住触发键时循环执行多个宏指令序列   |
| 跟随   | 按键按下时按下指定键，弹起时弹起指定键 |
| 组合   | 根据辅助键状态执行不同分支       |
| 映射   | 根据辅助键状态映射到不同按键      |
| 图像匹配 | 根据图像匹配结果执行不同分支      |
| 颜色匹配 | 根据颜色匹配结果执行不同分支      |
| 文字识别 | 根据文字识别结果执行不同分支      |

### 宏指令语法

| 指令 | 语法                          | 说明            |
| -- | --------------------------- | ------------- |
| 单击 | `按键名` 或 `MLeft x y`         | 单击键盘按键或鼠标指定位置 |
| 按下 | `按下 按键名` 或 `按下 MLeft x y`   | 按下键盘按键或鼠标指定位置 |
| 弹起 | `弹起 按键名` 或 `弹起 MLeft x y`   | 弹起键盘按键或鼠标指定位置 |
| 移动 | `移动 x y [duration] [steps]` | 移动鼠标到指定位置     |
| 滚轮 | `滚轮 amount [x y]`           | 滚动鼠标滚轮        |
| 延迟 | `延迟 seconds`                | 延迟指定时间（秒）     |

### 鼠标按钮名称

| 按钮名称   | 说明    |
| ------ | ----- |
| MLeft  | 鼠标左键  |
| MRight | 鼠标右键  |
| Middle | 鼠标中键  |
| MSide1  | 鼠标侧键1 |
| MSide2  | 鼠标侧键2 |
