# AutoGame

本软件是一个基于 Python 的自动化工具，支持前后台键鼠、图色、文字识别等功能。

## 技术栈

- **后端**：Python 3.10+
- **前端**：Vue 3 + TypeScript + Vite + codemirror
- **键鼠操作**：autoxkit
- **图像匹配**：autoxkit
- **文字识别**：ONNX PaddleOCR
- **GUI框架**：PyWebView

## 安装与运行

### 环境要求

- Python >= 3.10
- Node.js >= 18.0.0

### 安装步骤

```bash
克隆项目: git clone https://github.com/YorickFin/AutoGame.get

安装后端依赖: uv sync

安装前端依赖:
            cd frontend
            pnpm install

启动前端开发服务器: pnpm run dev

启动后端程序: uv run main.py
```

### 打包部署

```bash
构建前端:
        cd frontend
        pnpm run build

打包为可执行文件: pyinstaller main.spec -y
```
